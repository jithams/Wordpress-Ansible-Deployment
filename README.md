# WordPress Deployment on Amazon Linux 2023 Using Ansible

## Project Overview

This project demonstrates the deployment of a WordPress website on an Amazon Linux 2023 EC2 instance. The project includes the installation and configuration of Nginx, PHP, MariaDB, phpMyAdmin, SFTP, and SSL certificate for secure website hosting.

## Technologies Used

- AWS EC2
- Amazon Linux 2023
- Ansible
- Nginx
- PHP 8+
- MariaDB
- WordPress
- phpMyAdmin
- SFTP
- Certbot SSL

## Project Features

- WordPress website deployment
- MariaDB database configuration
- Nginx web server setup
- phpMyAdmin integration
- SFTP user access configuration
- Domain name configuration
- SSL certificate installation using Let's Encrypt
- Personal blog creation using WordPress
